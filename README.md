### About

Software engineer with a focus on **AI-driven automation** and **full-stack web systems**. I work primarily at the intersection of Computer Vision and backend engineering — building systems that automate processes which are currently manual, inconsistent, or unscalable.

My primary project is **ECR (E-Challan Relay)** — a traffic enforcement platform that uses YOLOv8 for real-time license plate recognition, validates vehicle documents against a central dataset, and dispatches e-challans via SMTP, all without manual intervention. The system is built across three services: React frontend, Node.js/Express backend, and a Python/Flask ML service.

I'm interested in **open-source civic tech**, MLOps, and building things that work in production — not just demos.

📍 Bangalore, India &nbsp;|&nbsp; 🎓 PES University &nbsp;|&nbsp; 📬 [LinkedIn](https://www.linkedin.com/in/himanshupandey04)
